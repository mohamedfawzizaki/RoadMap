#                                       AWS Data Transfer pricing model

## üßæ What Is AWS Data Transfer?

**Data transfer** in AWS refers to **any movement of data**:

* Between AWS services
* Across regions or Availability Zones
* In or out of AWS (internet)
* Across accounts, VPCs, or networks

> üß† Data transfer pricing is **not based on requests or storage**, but **on the volume of data (GBs) moved between locations.**

---

## üîç Types of Data Transfer

Let‚Äôs break this down into **6 major categories**:

---

### 1. **Data Transfer IN to AWS (from Internet)**

| From     | To                    | Cost                    |
| -------- | --------------------- | ----------------------- |
| Internet | EC2, S3, Lambda, etc. | **Free** in all regions |

> ‚úÖ You can ingest data into AWS (uploads, API requests) without any charges.

---

### 2. **Data Transfer OUT from AWS to Internet**

| From                    | To       | Cost                    |
| ----------------------- | -------- | ----------------------- |
| EC2 / S3 / ELB / Lambda | Internet | Charged per GB (tiered) |

#### üåê Typical Pricing (US East ‚Äì Virginia):

| Tier             | Price          |
| ---------------- | -------------- |
| First 1 GB/month | **Free**       |
| Next 10 TB       | **\$0.09/GB**  |
| Next 40 TB       | **\$0.085/GB** |
| Next 100 TB      | **\$0.07/GB**  |
| 150 TB+          | **\$0.05/GB**  |

> ‚úÖ Use **CloudFront**, **S3 Transfer Acceleration**, or **PrivateLink** to reduce these costs.

---

### 3. **Data Transfer Between AWS Services (Same Region)**

#### a. **Same Availability Zone (AZ)**

| From                            | To   | Cost |
| ------------------------------- | ---- | ---- |
| EC2 ‚Üí EC2 (same AZ, private IP) | Free |      |
| EC2 ‚Üí RDS (same AZ, private IP) | Free |      |
| Lambda ‚Üí S3 (same AZ)           | Free |      |

#### b. **Different AZs in Same Region**

| From                    | To                                   | Cost |
| ----------------------- | ------------------------------------ | ---- |
| EC2 (AZ A) ‚Üí EC2 (AZ B) | \~\$0.01/GB (both directions billed) |      |

> ‚ùóCross-AZ communication = billed **per GB, in both directions**

---

### 4. **Data Transfer Between Regions**

| From                              | To                                          | Cost |
| --------------------------------- | ------------------------------------------- | ---- |
| EC2 (us-east-1) ‚Üí EC2 (eu-west-1) | \~\$0.02/GB ‚Äì \$0.09/GB depending on region |      |

* Both directions are **charged separately**
* Applies to:

  * S3 replication
  * RDS cross-region
  * EC2 inter-region
  * Lambda cross-region

> üí° Use **Amazon CloudFront** or **AWS Global Accelerator** to optimize global latency and cost.

---

### 5. **Data Transfer to/from CloudFront**

| Direction                         | Cost                            |
| --------------------------------- | ------------------------------- |
| Origin (S3/EC2) ‚Üí CloudFront edge | **Free**                        |
| CloudFront ‚Üí User (Internet)      | Much cheaper than EC2/S3 direct |

> CloudFront acts as a **cost-effective CDN** to reduce data transfer costs out to users.

---

### 6. **VPC Peering, Transit Gateway, PrivateLink**

| Type                           | From/To             | Cost                                           |
| ------------------------------ | ------------------- | ---------------------------------------------- |
| **VPC Peering** (same region)  | VPC A ‚Üî VPC B       | \~\$0.01/GB                                    |
| **VPC Peering** (cross-region) | VPC A ‚Üî VPC B       | \~\$0.02‚Äì\$0.09/GB                             |
| **Transit Gateway**            | Any VPC via TGW     | \~\$0.02/GB +                                  |
| \$0.05 per hour/attachment     |                     |                                                |
| **PrivateLink**                | Producer ‚Üî Consumer | \~\$0.01‚Äì\$0.02/GB                             |
| **Site-to-Site VPN**           | On-prem ‚Üî AWS       | \~\$0.05/GB                                    |
| **Direct Connect**             | On-prem ‚Üî AWS       | Based on port + transfer (as low as \$0.02/GB) |

## üß† Common Data Transfer Cost Pitfalls

| Problem                           | Impact                                            |
| --------------------------------- | ------------------------------------------------- |
| üîÅ **Cross-AZ traffic**           | Adds unexpected \$0.01/GB in high-throughput apps |
| üåç **Cross-region replication**   | Adds \$0.02‚Äì\$0.09/GB per direction               |
| üåê **EC2 to Internet**            | \$0.09/GB is costly at scale                      |
| üö´ **Public IP usage** inside AWS | Billed as **internet transfer**, not internal     |

> üìå Always use **private IPs**, **same AZ**, and **CloudFront** where possible.

## üìä Example Scenario

You run:

## üì¶ Tips to Reduce Data Transfer Costs

| Tip                                      | Benefit                                              |
| ---------------------------------------- | ---------------------------------------------------- |
| Use **CloudFront CDN**                   | Cache at edge; reduce origin transfer                |
| Use **S3 Transfer Acceleration**         | Speeds + reduces transfer charges for global uploads |
| Use **PrivateLink** or **VPC Endpoints** | Avoid internet/public IPs for service comms          |
| Keep services in **same AZ**             | Avoid \$0.01/GB cross-AZ fees                        |
| Use **Global Accelerator**               | Efficient multi-region routing for latency + cost    |
| **Compress data**                        | Reduces payload volume transferred                   |
