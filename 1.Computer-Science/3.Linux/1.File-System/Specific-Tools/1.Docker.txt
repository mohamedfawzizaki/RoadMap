                                                 Docker Files in Linux

When Docker is installed on a Linux system, it stores various files and directories related to 
 its configuration, images, containers, volumes, and logs.
Below is a detailed list of where Docker files are typically located:
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
1.Docker Configuration Files

    - System-wide Configuration File:
      - `/etc/docker/daemon.json`  
        - This file contains the Docker daemon's configuration.
        - You can customize options such as the default storage driver, logging configuration, and more.

    - Service Files:
      - `/lib/systemd/system/docker.service`  
        The systemd service file that manages the Docker daemon.

    - Default Environment File:
      - `/etc/default/docker`  
        Contains default options passed to the Docker daemon at startup.
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
2.Docker Images, Containers, Volumes, and Networks

    - Default Docker Storage Location:
      - `/var/lib/docker/`  
        This is the primary directory where Docker stores all data, including:
      - `/var/lib/docker/containers/` – Stores container-specific data, such as logs.
      - `/var/lib/docker/images/`     – Stores Docker images.
      - `/var/lib/docker/volumes/`    – Stores Docker volumes.
      - `/var/lib/docker/network/`    – Stores network information for Docker networks.

    - Checking the Docker Root Directory:
      If you want to verify or change the default Docker root directory (`/var/lib/docker`),
      you can inspect or modify it in the `daemon.json` file:
      ```bash
      cat /etc/docker/daemon.json
      ```
      Example of changing the root directory:
      ```json
      {
        "data-root": "/custom/docker/data"
      }
  ```
---------------------------------------------------------------------------------------------------------
3.Docker Logs

    - Container Logs:  
      Docker container logs are stored in:
      - `/var/lib/docker/containers/<container-id>/<container-id>-json.log`  
        Each container has its own log file in JSON format.
      
    - Daemon Logs:  
      The Docker daemon logs are managed by the system’s logging service:
      - Systemd-based systems:  
        ```bash
        journalctl -u docker.service
        ```

---------------------------------------------------------------------------------------------------------
4.Docker CLI Configuration

    - User-specific Configuration File:  
      The Docker CLI stores its configuration for individual users in:
      - `~/.docker/config.json`  
        This file contains credentials for registries and CLI preferences.

---------------------------------------------------------------------------------------------------------
5.Docker Compose Files

    - Docker Compose Files:  
      These are user-defined files and can be located anywhere.They are typically named `docker-compose.
      yml` and stored in project directories.

---------------------------------------------------------------------------------------------------------
6.Temporary Files

    - Temporary Socket and PID Files:  
      Docker stores temporary files in:
      - `/var/run/docker.sock` – The Unix socket used by the Docker CLI to communicate with the Docker daemon.
      - `/var/run/docker.pid` – The PID file for the running Docker daemon.

---------------------------------------------------------------------------------------------------------
Useful Docker Commands to Inspect File Locations

    1.Check Docker Info
      ```bash
      docker info
      ```
      This command displays the current root directory (`/var/lib/docker`) and storage driver in use.

    2.List All Containers
      ```bash
      docker ps -a
      ```

    3.List All Images
      ```bash
      docker images
      ```

    4.List All Volumes
      ```bash
      docker volume ls
      ```
