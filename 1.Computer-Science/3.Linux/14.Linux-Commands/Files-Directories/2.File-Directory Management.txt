                                                  Directory Management
                                                  ==================== 
         __________________________________________________________________________________________________________________________________________________
        | Command  |              Example                                 |                                        Usage                                                               
        |__________|______________________________________________________|________________________________________________________________________________         
        |  ls      |   ls [opt]  <dest-dir>                               | List directory contents.
        |  tree    |   tree      <dest-dir>                               | Display directory contenet recursively.
        |  pwd     |   pwd                                                | Print working directory.
        |  file    |   file      <dest-file/dir>                          | Determine file or dir type.
        |  stat    |   stat      <dest-file/dir>                          | Display file or file system status.
        |--------------------------------------------------------------------------------------------------------------------------------------------------
        |  cd      |   cd        <dest-dir>                               | Change directory.
        |--------------------------------------------------------------------------------------------------------------------------------------------------
        |  mkdir   |   mkdir     <dir-name>                               | Create a new directory.
        |  rmdir   |   rmdir     <dir-name>                               | Remove an empty directory.
        |  rm      |   rm -rf    <dir-name>                               | Remove files or directories.
        |--------------------------------------------------------------------------------------------------------------------------------------------------
        |  cp      |   cp        <source-dir> <dest-dir>                  | Copy files or directories.
        |  mv      |   mv        <source-dir> <dest-dir>                  | Move or rename files or directories.
        |__________|______________________________________________________|________________________________________________________________________________
